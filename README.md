## 2026 Data Flow — Event-Driven Architecture (EDA)

### overview
경험(Sources)이 이벤트를 만들고, 이벤트는 스트림을 타고 흐르며, ‘멈춤/해석/경계’로 반응을 설계하고, 결과를 저장하고, 관측으로 다시 조정한다.

“경험은 쌓이고, 사건은 흐르고, 올해도 응원한다.”
